PyQt App Shell — Готовий до збірки застосунок

Цей проєкт містить мінімальний, але повністю працездатний графічний застосунок на PyQt6, який можна запускати у режимі розробки або збирати в exe / standalone-бінарний файл за допомогою PyInstaller, демонструє готовий до використання GUI, інструкції зі збірки та структуру проєкту.
Що робить застосунок
У вікні доступні:

1.Текстовий редактор
2.Поле введення
3.Кнопка Save notes to file — записує текст у файл
4.Кнопка Open text file — відкриває .txt
5.Кнопка About — показує інформацію про застосунок
6.Статус-бар з повідомленнями

Все реалізовано на PyQt6.QtWidgets.
Додаткові поради для перевіряючих:

1.Для Windows рекомендується збірка на Windows (крос-компіляція з Linux не підтримується).
2.Якщо додасте нові ресурси (іконки, .ui, зображення) — потрібно буде додати їх у PyInstaller через --add-data.
